# 📖 About

**PS2 Loop Wallpaper Pack** is a collection of high-quality wallpapers inspired by some of the most iconic PlayStation 2 screens.

Each wallpaper is available as both a static image and a seamless looping video, making it suitable for game launchers, emulators, frontends, media centers, desktop customization software, and any application that supports custom image or video backgrounds.

The project is continuously expanded with new wallpaper collections over time.
